# Cinema

O módulo de **Cinema** permite que o cliente compre ingressos e acompanhe a programação dos cinemas parceiros sem sair do app. Existem dois fluxos disponíveis, que podem ser usados de forma independente ou combinada:

- **[Seleção de Cadeiras](/docs/modulos/cinema/selecao-cadeiras)** — o cliente escolhe o assento interativamente no mapa da sala (disponível para Cineart e Grupo Cine)
- **[Compra por Código](/docs/modulos/cinema/compra-por-codigo)** — o cliente compra o ingresso informando um código de sessão ou voucher

## Disponível para

- Todos os clientes com kit básico ou acima.
