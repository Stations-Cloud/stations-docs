# Compra de Ingresso por Código

O fluxo de **Compra por Código** permite que o usuário compre um voucher com desconto — ideal para cinemas que não possuem integração direta ou para campanhas com códigos promocionais.

## O que o usuário pode fazer

- Comprar ingressos com até **60% de desconto**  
- Utilizar o ingresso em **qualquer dia, horário ou unidade da rede**  
- Receber um **código digital** no app para uso no site ou no cinema  

## Como funciona

1. O usuário escolhe a rede de cinema no app  
2. Realiza a compra do ingresso com desconto  
3. Recebe um **código via push notification**  
4. Utiliza esse código de duas formas:
   - No **site do cinema**, escolhendo filme, data e horário  
   - Diretamente no cinema (bilheteria ou totens de autoatendimento)

## Quando esse fluxo é utilizado

Este fluxo é indicado para:

- Cinemas **sem integração com mapa de sala**  
- **Campanhas promocionais** com vouchers (ex: benefícios corporativos, clubes de assinatura)  

## Disponibilidade

Todas as empresas com o **kit básico** já têm acesso a este módulo.
