# Documentação do Super App Stations

Esta documentação existe para ser a **fonte única de verdade** sobre o Super App Stations — para clientes, parceiros e times internos.

Aqui você encontra tudo o que precisa para entender, operar e acompanhar o app:

- **O que cada módulo faz** e como ele funciona para o usuário final
- **Quais integrações estão disponíveis** e com quais sistemas o app se conecta
- **Regras de negócio** — quem tem acesso a quê, quais módulos dependem de contrato, quais estão inclusos no kit básico
- **Controle de versões** — o que mudou em cada atualização, quando foi lançado e como identificar a versão instalada no celular do usuário
- **Configurações e comportamentos** do app que impactam a experiência do cliente

O app é uma plataforma modular — cada empresa que usa a Stations pode ter uma composição diferente de funcionalidades. Esta documentação cobre todos os módulos disponíveis, cabendo a cada empresa identificar o que está ativo no seu app.

## Módulos disponíveis

| Módulo | O que oferece |
|--------|--------------|
| [Cashback](/docs/modulos/cashback) | Dinheiro de volta em compras |
| [Descontos e Benefícios](/docs/modulos/desconto) | Cupons e ofertas via Alloyal, Hubconn, Du Benefícios ou rede própria |
| [Gás](/docs/modulos/gas) | Compra e entrega de gás |
| [Academia](/docs/modulos/academia) | Gestão de contratos e treinos em academias |
| [Carteira Digital](/docs/modulos/carteira) | Saldo, extrato e recarga |
| [Internet (ISP)](/docs/modulos/internet) | Gerenciamento de planos de internet |
| [Telefonia (MVNO)](/docs/modulos/telefonia) | Planos de celular e recargas |
| [Cinema](/docs/modulos/cinema) | Compra de ingressos — seleção de cadeiras ou por código |
| [Gift Card](/docs/modulos/gift-card) | Compra e resgate de cartões presente |
| [Postos](/docs/modulos/postos) | Abastecimento em postos parceiros |
