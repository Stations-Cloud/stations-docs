# Cashback

O módulo de **Cashback** permite que os clientes acumulem dinheiro de volta a cada compra ou interação no aplicativo, resgatando o saldo diretamente na carteira digital.

## O que o cliente pode fazer

- Ver o saldo de cashback disponível na tela inicial
- Acompanhar o histórico de cashback recebido por transação
- Resgatar o cashback acumulado para a carteira digital
- Ver detalhes de cada lançamento (data, valor, origem)

## Como funciona

O cashback é creditado automaticamente após ações elegíveis — como compras em parceiros, pagamento de faturas ou uso de serviços no app. O saldo fica disponível para uso conforme as regras da empresa contratante.

## Disponível para

Empresas que desejam oferecer um programa de recompensas financeiras aos seus clientes.
